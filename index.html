📋 1. Visão geral

A extensão Suporte Tools (ID: oinhbadjkedncidapilndkokkjepglnc) foi desenvolvida para aumentar a produtividade de técnicos de provedores de internet (ISPs). Ela automatiza o login em roteadores locais ZTE e otimiza a leitura e o preenchimento de notas técnicas dentro do sistema de gestão SGP.

🛡️
Compromisso com a privacidade: a Suporte Tools foi projetada com privacidade desde o início. Nenhum dado pessoal, de navegação ou credencial é enviado a servidores externos. Toda a lógica é executada localmente no dispositivo do técnico.

🗄️ 2. Dados coletados

A tabela abaixo detalha cada categoria de dado e como ela é tratada:

Categoria Coletado? Destino
Credenciais de roteador (usuário/senha) Somente local chrome.storage.local — nunca sai do dispositivo
Informações de identificação pessoal Não coletado —
Histórico de navegação Não coletado —
Dados financeiros ou de saúde Não coletado —
Localização ou endereço IP Não coletado —
Atividade do usuário / teclas pressionadas Não coletado —
Conteúdo de páginas visitadas Não transmitido Lido em tempo de execução apenas para preencher formulários SGP/ZTE
🚫
Nunca vendemos, alugamos ou compartilhamos quaisquer dados do usuário com terceiros, parceiros comerciais ou anunciantes, em nenhuma circunstância.

💾 3. Armazenamento local

A extensão utiliza exclusivamente a API chrome.storage.local para persistir os perfis de acesso a roteadores (endereço IP, usuário e senha) que o técnico cadastra voluntariamente. Esses dados:

ℹ️
Ficam 100% no dispositivo local — não são sincronizados com a conta Google, não são enviados a nenhum servidor e não são acessíveis por outros sites ou extensões.

O usuário pode remover todos os dados armazenados a qualquer momento através do popup da extensão ou desinstalando a extensão, o que apaga automaticamente todos os dados do chrome.storage.local.

🔑 4. Permissões solicitadas

A extensão solicita apenas as permissões estritamente necessárias para sua funcionalidade:

Permissão Finalidade
storage Salvar perfis de roteadores localmente para acesso rápido pelo técnico
http://*/* e https://*/* Roteadores ZTE operam em IPs privados arbitrários (ex.: 192.168.x.x, 10.x.x.x) não previsíveis em tempo de desenvolvimento. O painel SGP varia por provedor. A extensão detecta a estrutura específica dos formulários antes de agir.

🌐 5. Terceiros

A Suporte Tools não integra serviços de análise, rastreamento, publicidade ou qualquer biblioteca de terceiros que transmita dados do usuário para fora do navegador. Não há requisições para servidores externos originadas pela extensão.

⚖️ 6. Direitos do usuário

Como os dados ficam integralmente no dispositivo do próprio usuário, o controle é total:

✅
Acesso e exclusão: abra o popup da extensão e remova individualmente qualquer perfil salvo, ou desinstale a extensão para apagar todos os dados de uma só vez.

A extensão não coleta dados que exijam solicitação formal de exclusão a terceiros.

🔄 7. Alterações nesta política

Esta Política de Privacidade pode ser atualizada conforme novas versões da extensão forem lançadas. Sempre que houver alteração relevante, a data de "última atualização" no topo desta página será modificada. A premissa de não coletar dados externos será mantida em todas as versões futuras.
